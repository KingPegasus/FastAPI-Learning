# Running Procedure

## Pre-requisites

- Install Python 3.7+
- Configure virtual environment using link in Getting Started

## Install Dependencies

- pip install fastapi
- pip install "uvicorn[standard]"

## Run in terminal

- cd Projectx
- uvicorn nameoffile:app --reload

## Running Swagger UI

- http://127.0.0.1:8000/docs
